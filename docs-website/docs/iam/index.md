# IAM Infrastructure

## Overview

Manage identity and access with Access Groups, Service IDs, and fine-grained policy controls.

## 💡 Key Features

- **Access Groups**: Organize users and service IDs
- **Service IDs**: Non-human identities for automation
- **Policies**: Fine-grained access control
- **API Keys**: Secure authentication

## 🎯 Use Cases

User management, service authentication, role-based access, and automation.
